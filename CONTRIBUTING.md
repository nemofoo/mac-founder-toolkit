# Contributing

Thanks for helping grow this list! Here's how to add a tool.

## Guidelines

- **Mac-native or Mac-first tools only.** Cross-platform is fine if the Mac experience is great.
- **One tool per PR.** Makes it easier to review.
- **Add to the bottom** of the relevant category table.
- **Include all columns:** Tool (with link), Description (one sentence), Pricing (Free / Paid / Free + Paid).
- **No affiliate links.**
- **Check for duplicates** before submitting.

## How to submit

1. Fork this repo
2. Add your tool to the relevant section in `README.md`
3. Open a pull request with the tool name as the title

## New categories

If your tool doesn't fit an existing category, propose a new one in your PR description. Keep it broad enough that others could add to it.
